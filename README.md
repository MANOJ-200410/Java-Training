Given the number of rows and columns, print a rectangle pattern using the * symbol.
Input Format

Two integers separated by space: rows and columns
Constraints

1≤rows,columns≤100
Output Format

Print a rectangle of rows × columns made of * symbols.
Sample Input 0

3 5
Sample Output 0

*****
*****
*****
Sample Input 1

2 4
Sample Output 1

****
****
